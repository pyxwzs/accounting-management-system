# 个人记账管理系统 — 技术栈说明

## 总体架构

前后端分离：**Vue 3 + Element Plus** 管理端/前台，**Spring Boot** 提供 REST API，**MySQL** 持久化，**Redis** 辅助缓存，**Apache Shiro + JWT** 做认证与鉴权。

---

## 后端（`admin-vue-api`）

| 类别 | 技术 | 版本/说明 |
|------|------|-----------|
| 语言与构建 | Java 8、Maven | `pom.xml` |
| 核心框架 | Spring Boot | 2.7.11 |
| Web | Spring MVC、`spring-boot-starter-web` | REST 接口 |
| 持久层 | MyBatis-Plus | 3.5.3.1 |
| 数据库 | MySQL、mysql-connector-java | JDBC 5.1.42 |
| 连接池 | Druid | `druid-spring-boot-starter` 1.1.9 |
| 安全 | Apache Shiro、`shiro-spring` | 1.11.0；配合自定义 `JwtFilter`、`CustomRealm` |
| Token | java-jwt (Auth0) | 4.4.0 |
| 缓存 | Spring Data Redis | `spring-boot-starter-data-redis` |
| AOP | Spring AOP | 操作日志 `LogAspect` |
| 校验 | Bean Validation | `spring-boot-starter-validation` |
| JSON | Fastjson | 1.2.47 |
| 邮件 | Spring Mail | 预算预警等场景 |
| API 文档 | Swagger2、swagger-bootstrap-ui | 2.9.2 / 1.9.6 |
| 工具 | Lombok | 简化实体与日志 |
| 其他 | emoji-java | 文本过滤；定时任务 `@EnableScheduling` |

**运行与配置**：`application.yml`（端口、数据源、MyBatis-Plus、Shiro JWT、QQ 邮箱、`finance.rss` 等）。

---

## 前端（`vue-elementplus`）

| 类别 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 | Composition API |
| 构建 | Vite 4 | `npm run dev` / `build` |
| UI | Element Plus 2.x | 表格、表单、`el-tabs` 等 |
| 路由 | Vue Router 4 | 含收件箱、重定向等 |
| 状态 | Pinia + persistedstate | 持久化登录态等 |
| HTTP | Axios 1.x | 拦截器携带 `Authorization` |
| 图表 | ECharts、vue-echarts | 预算饼图等 |
| 编辑器 | @wangeditor/editor-for-vue | 富文本 |
| 其他 | nprogress、html2canvas、jspdf、xlsx、地区数据等 | 进度条、导出、地区选择 |

## 数据库与脚本

- 业务表与初始化数据：`database/obt_db.sql`（含 `sys_budget`、`sys_article_favorite`、`sys_log` 等）。

---

## 开发与部署提示

- 后端默认端口 **8080**；前端开发时代理或直连该地址需与部署一致。
- 生产环境请替换 `application.yml` 中的数据库密码、JWT 密钥、邮箱授权码等敏感配置，勿提交真实密钥到版本库。
